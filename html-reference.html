Reference Page for Common HTML Elements

Element Name Syntax Description
HTML <html></html> The <html> tag deliniates an HTML file, and so must be placed at the beginning of every file and at the end of every file.
Header <head></head> The <head> tage follows the <html> tag and defines the header of a page. This is where metadata (non-graphical) is stored, like the title and styles.
Title <title></title> The space in between the <title> and </title> tags becomes the title of the page, which is displayed at the top.
Body <body></body> The <body> tags define the start and end of the physical content ie. text and images that will be displayed.
Headings <h1></h1> to <h6></h6> The number in the tag defines the size of the header, while the text in between becomes the header.
Paragraphs <p></p> The text in between the <p> tags becomes a new paragraph.
Breaks <br> The break starts with <br> with no closing tag. It creates a break in the text.
Lines <hr>; <hr> has no closing tag, and signals a change in thematic content
Comments <!-- --> The content in between <!-- --> doesn't affect the content in the page and is only for the developers.
Links <a></a> The href tag in the first <a> tag defines the address for the link, and the space in between the two tags defines the name of the link.
Images <img></img> The src tag is the source for the image inside the first <img> tag, and the alt tag provides alternative text.
Lists <ul><ol><li> The <ul> defines an unordered list, and the <ol> list defines an ordered list. <li> defines the items being listed.
Tables <table><th></th><tr><td></td></tr></tab;e> The <table> tag is the beginning of the table. The <th> tag defines the table headers. The <tr> tag defines rows, and the <td> tag defines individual elements.
Blocks <div></div> The <div> tags seperate the HTML code into sections, allowing for easier application of CSS to certain parts of the page.

Remember, the <!DOCTYPE>, <html>, <head>, and <body> tags are essential in every single HTML page.

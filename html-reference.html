Reference Page for Common HTML Elements

HTML

Description

The <html> tag deliniates an HTML file, and so must be placed at the beginning of every file and at the end of every file.

Syntax

<html></html>

Example

This is non-graphical.

Header

Description

The <head> tage follows the <html> tag and defines the header of a page. This is where metadata (non-graphical) is stored, like the title and styles.

Syntax

<head></head>

Example

This is non-graphical.

W3 Schools Link

Title

Description

The space in between the <title> and </title> tags becomes the title of the page, which is displayed at the top.

Syntax

<title></title>

Example

The title of this page has been set to HTML Reference Page.

W3 Schools Link

Body

Description

The <body> tags define the start and end of the physical content ie. text and images that will be displayed.

Syntax

<body></body>

Example

All of this information is contained inside of the body.

Headings

Description

The number in the tag defines the size of the header, while the text in between becomes the header.

Syntax

<h1></h1> to <h6></h6>

Example

This is heading size 6.

Description

The text in between the <p> tags becomes a new paragraph.

Syntax

Links

Description

The href tag in the first <a> tag defines the address for the link, and the space in between the two tags defines the name of the link.

Syntax

<a></a>

Example

This is a link.

W3 Schools Link

Images

Description

The src tag is the source for the image inside the <img> tag, and the alt tag provides alternative text. There is no closing tag.

Syntax

<img>

Example

This is an image.

W3 Schools Link

Lists

Description

The <ul> defines an unordered list, and the <ol> list defines an ordered list. <li> defines the items being listed.

Syntax

<ul><ol><li>

Example

  1. This is an
  2. ordered list

W3 Schools Link

Tables

Description

The <table> tag is the beginning of the table. The <th> tag defines the table headers. The <tr> tag defines rows, and the <td> tag defines individual elements.

Syntax

<table><th></th><tr><td></td></tr></table>

Example

This reference page has a table at the top.

W3 Schools Link

Blocks

Description

The <div> tags seperate the HTML code into sections, allowing for easier application of CSS to certain parts of the page.

Syntax

<div></div>

Example

A block on its own is non-graphical.
